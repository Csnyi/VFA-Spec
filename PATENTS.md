# Patent Notice

The Virtual Flow Agreement (VFA) specification in this repository is
published to support open research, public review, and interoperable
implementations of the protocol.

Independent implementations of the VFA protocol for research,
experimentation, and interoperability are intended to be allowed.

The VFA concept described in this repository may be subject to
future patent applications.

Any such filings would aim to protect the conceptual mechanism while
preserving the ability for independent implementations of the protocol
specification.

Implementations based on this specification are permitted under the
terms of the Apache License 2.0 included in this repository.

This repository provides a draft specification and reference materials
for the VFA architecture.
